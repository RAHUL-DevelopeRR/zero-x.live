# Design System Document

This document outlines the core design principles and specifications for our user interfaces.

## Theme Overview

Our design system utilizes a **light** color mode, prioritizing clarity and visibility for a comfortable user experience.

## Color Palette

The color palette is built around a vibrant green foundation, with a warm tertiary accent, providing a balanced yet energetic feel.

- **Primary Color**: `#0fff46` - A bright, energetic green used for key interactive elements, calls to action, and primary branding.
- **Secondary Color**: `#0e8a23` - A deeper, more grounded green supporting less prominent UI elements, chips, and secondary actions.
- **Tertiary Color**: `#ffd3ab` - A soft, warm accent color for highlights, badges, or decorative elements, adding a touch of friendliness.
- **Neutral Color**: `#6f7a6a` - A balanced neutral for backgrounds, surfaces, and non-chromatic elements, ensuring readability and focus.

## Typography

Our typography system uses the `Inter` font family consistently across all text elements to maintain a cohesive and modern aesthetic.

- **Headline Font**: `Inter` - Used for all headings, ensuring strong visual hierarchy.
- **Body Font**: `Inter` - Applied to all paragraph text and general content for optimal readability.
- **Label Font**: `Inter` - Utilized for form labels, buttons, and other interactive text elements.

## Shape and Spacing

### Roundedness

UI elements feature a **moderate (2)** level of roundedness, providing a friendly and approachable feel without being overly soft. This strikes a balance between modern aesthetics and usability.

### Spacing

The design system employs **normal (2)** spacing, ensuring a balanced amount of whitespace around and between UI elements. This provides sufficient visual breathing room, enhancing readability and content organization without feeling too sparse or too dense.
