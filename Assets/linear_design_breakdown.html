Linear.app landing page design breakdown — every element analyzed

1
Visual Foundation & Tone
ThemePure dark. Background is near-black (#0f0f12), not true black — gives depth without harshness
MoodEngineering-grade precision. Silent confidence. No noise, no decoration for its own sake
Principle"Designed to the last pixel, engineered with unforgiving precision" — every element earns its place
TextureSubtle grain/noise overlay on the hero — adds depth without breaking the dark theme
2
Color System
Background#0f0f12 — base canvas
Surface#1a1a1f — cards, panels, UI chrome
Borders#2a2a30 — ultra-subtle, 1px lines only
Text primary#f0f0f0 — off-white, not pure white (prevents harshness)
Text secondary#888896 — muted labels, captions
Accent#5e6ad2 — Linear purple. Used sparingly on CTAs and active states only
Glow blobsRadial gradient "light sources" — purple/violet orbs at 10–15% opacity, blurred 120px+. Create depth illusion behind hero
3
Typography
FontInter exclusively — the entire page. Chosen to feel "invisible" — like reading code, not marketing
Hero headline56–64px, weight 500 (not bold). Line-height 1.1. Tight tracking (-0.02em). 2–3 lines max
Hero subhead18–20px, weight 400, color #888896. Max 2 lines. Never more than 15 words
Body / features15–16px, weight 400, line-height 1.6. Section headers 13px uppercase, letter-spacing 0.08em, muted
HierarchyWeight contrast only — no size jumps. 500 vs 400 does all the work
4
Navigation Bar
StructureFull-width, sticky. Logo left · Links center · CTA right
StyleTranslucent dark background with backdrop-filter blur. 1px bottom border. Height ~56px
LogoMark (circle with partial fill) + wordmark "Linear" in 500 weight. Pure white
Nav links5–6 items, 14px, weight 400, #888 default → white on hover. No underlines. No dropdown
CTA button"Sign up" — white bg, dark text, 32px height, 8px radius. Compact. Never purple in nav
5
Hero Section
LayoutFull-viewport centered column. Text block top, product screenshot below
EyebrowPill badge above headline — e.g. "New: Linear Agent" with subtle purple border. 12px, rounded-full
Headline"Linear is shaped by the practices and principles of world-class product teams" — Statement of belief, not a feature list
CTA clusterTwo buttons side by side: primary (white filled) + secondary (ghost/outline). Gap 12px
Social proof"Powers over 25,000 product teams" — 1 line, muted, below CTA. No logos in hero
Hero visualActual app UI screenshot in a dark browser frame. Floats with subtle shadow. Shows real product
Background fx2–3 blurred radial gradient orbs (purple, blue) at very low opacity behind hero text
6
Feature Sections
PatternAlternating: text left + visual right, then text right + visual left. Clean 2-col grid
Section label13px uppercase muted tag above the headline — e.g. "ISSUES" or "CYCLES". Tiny eyebrow
Feature headline28–32px, weight 500, 2 lines max. Direct noun phrase — not a question
Feature copy2–3 sentences max. 16px, muted. Specific, never vague marketing speak
VisualsDark UI mockups in bordered containers. Keyboard shortcut overlays. Animated micro-interactions
Spacing120px+ between sections. Enormous whitespace is the visual rhythm
7
Social Proof / Testimonials
Format3-column card grid. Dark surface cards (#1a1a1f). Quote in 15px italic. Author name + role below
QuotesShort, punchy, specific. "You just have to use it and you will see, you will just feel it." — no fluff
Company logosShown as a horizontal scrolling strip above or below testimonials — white/muted, not colored
Stat block"25,000+ teams" — large number, small label below. Used as standalone trust element
8
Interactive Product Demo (live UI embed)
What it isActual animated product UI embedded in the page. Shows real-time issue creation, keyboard shortcuts, command palette
StyleRuns inside a rounded dark frame (simulated browser or app window). No fake screenshots
MotionSmooth 60fps animations. Text appears typed out. Status badges update. Feels alive
PurposeProve the product before signup. "50ms interactions" — you experience it, not read about it
9
Bottom CTA / Footer
CTA sectionFull-width dark panel. Centered headline "Start building with Linear" + 2 buttons. Same orb glow behind
Footer4-col link grid: Product / Company / Resources / Legal. 13px muted links. Logo + tagline left
Footer border1px top border only. Absolutely no footer background color change
10
Motion & Micro-interactions
Scroll revealsElements fade-up on scroll. 300ms ease-out. Never dramatic — subtle and fast
Hover statesLinks: color shift 150ms. Buttons: subtle scale(1.01) + brightness. Cards: border brightens
No parallaxZero parallax scrolling. No scroll-jacking. Fast page, native scroll speed always
Gradient orbsSlowly drifting radial blobs. CSS keyframe animation, very slow (20–40s loop). Purely ambient
