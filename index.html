Flagship Product

LuminaPlayer

An AI-powered media player that generates real-time subtitles, detects audio events, and translates content — all offline.

Intelligent media playback,
redefined.

LuminaPlayer integrates cutting-edge AI models into a seamless desktop experience. No cloud dependency, no subscriptions — pure on-device intelligence.

  • Real-time subtitle generation via Whisper ASR
  • 521-class audio event detection with YAMNet
  • Offline translation via Argos Translate
  • Parallel multi-threaded pipeline architecture
  • Custom .dsrt subtitle format with binary search
