Capabilities
Why Neuron is different
It doesn't match keywords. It understands the semantic meaning behind your search using neural embeddings.

Semantic Search

Powered by all-MiniLM-L6-v2 neural embeddings. Search by meaning, not exact keywords.

Encyl AI Summarizer

Press Tab on any file for an instant AI summary. Uses llama3.2:1b via Ollama — 100% local.

Windows 11 Native

DWM Mica/Acrylic frosted glass, Segoe Fluent icons, native context menus.

Lightning Fast

FAISS vector index with HNSW graph search. Results in <5ms across 100,000+ files.

100% Offline

Everything runs on your machine. No cloud, no telemetry, no data ever leaves your PC.

40+ File Types

PDF, DOCX, PPTX, XLSX, Python, JavaScript, JSON, YAML, and dozens more.

Download & Install
