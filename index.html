Capabilities

Why Neuron is different

Neuron doesn't do keyword matching. It understands the semantic meaning behind your search query using neural embeddings.

🧠

Semantic Search

Powered by all-MiniLM-L6-v2 neural embeddings. Search "budget report" and find "Q3_financial_summary.xlsx" — even when the words don't match.

🤖

Encyl AI Summarizer

Select any file and press Tab to get an instant AI-generated summary. Uses llama3.2:1b via Ollama — 100% local, zero cloud.

🪟

Windows 11 Native

DWM Mica/Acrylic frosted glass, Segoe Fluent icons, native context menus. Feels like it's built into Windows.

⚡

Lightning Fast

FAISS vector index with HNSW graph search. Finds results in <5ms across 100,000+ files. Instant, no waiting.

🔒

100% Offline

Everything runs on your machine. No cloud, no telemetry, no data leaves your PC. Your files are your business.

📁

40+ File Types

PDF, DOCX, PPTX, XLSX, TXT, MD, Python, JavaScript, JSON, YAML, HTML, CSS, and dozens more — all indexed automatically.
